roomii

roomii is a concept for a mobile messaging application.

i worked with jacqy alqueza to design a messaging platform designed to improve communication for people living with housemates. the app allows users to chat, send money, plan events, and share notes. the design is motivated by my experience living with crappy rooommates and how things would have been better if communication was improved.

csv reader

csv reader is a web application to plot csv files.

i built a simple angular 6 application to load csv files and plot them on a multiseries line chart using chart.js. the design follows the material design guidelines and utilizes material design lite styles. the application features the ability to edit csv files directly on the webpage as well as export any changes made to the csv file.
