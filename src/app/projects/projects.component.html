palette

palette helps you find cool colors.

as an engineer that loves design, i sometimes have difficulty finding colors for my ux/ui projects. palette alleviates this issue by generating color palettes. the colors are generated by creating a single pastel color (i love pastel) and then averaging randomized rgb values to generate matching colors.

[Angular 6] [UI Design] [Sketch] [Principle]

yelp review classifier

yelp review classifier helps people choose their rating on yelp.

This project uses scikit-learn to classify text data from Yelp reviews to predict what rating will be given. The application first preprocesses the Yelp data. Then the application creates a K-Folds cross validator with 20 splits. A Bagging Classifier is initialized with 5 Logistic Regression models.

[Python 3] [Scikit Learn] [Flask] [NLTK] [Machine Learning]

csv reader

csv reader helps you visualize your csv files.

i built a simple angular 6 application to load csv files and plot them on a multiseries line chart using chart.js. the design follows the material design guidelines and utilizes material design lite styles. the application features the ability to edit csv files directly on the webpage as well as export any changes made to the csv file.

[Angular 6] [Chart.js] [UI Design] [Material Design] [Sketch]
